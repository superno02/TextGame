# 裝備與物品數值

本文件說明 TextGame 中所有物品的數值設計，包含武器、防具、消耗品、素材，以及裝備欄位系統與交易數值。

> 最後更新：2026-03-27（從 CombatStats.md 分拆）

---

## 一、物品數值設計

物品由 `items.json` 定義，分為六大類型。每件物品含攻擊力、防禦力、素質修正、使用條件等數值。

### 物品類型

| 類型 | 英文 | 說明 | 可裝備 | 可堆疊 |
|------|------|------|:------:|:------:|
| 武器 | `weapon` | 提供攻擊力，裝備於主手 | ✓ | ✗ |
| 防具 | `armor` | 提供防禦力，裝備於各部位 | ✓ | ✗ |
| 消耗品 | `consumable` | 使用後恢復狀態值 | ✗ | ✓ |
| 素材 | `material` | 怪物掉落的素材 | ✗ | ✓ |
| 任務物品 | `quest` | 任務相關物品 | ✗ | 依設定 |
| 雜物 | `misc` | 其他類型物品 | ✗ | 依設定 |

## 二、裝備欄位系統

角色共有 7 個裝備欄位：

| 欄位 | 英文 | 可裝備類型 | 目前可用裝備 |
|------|------|------------|------------|
| 頭部 | `head` | 防具 | 鐵盔 |
| 身體 | `body` | 防具 | 皮甲、鎖子甲、法師長袍 |
| 手部 | `hands` | 防具 | 皮手套 |
| 腿部 | `legs` | 防具 | 鐵脛甲 |
| 足部 | `feet` | 防具 | 布靴 |
| 主手 | `mainHand` | 武器 | 鐵劍、木杖、短弓、生鏽匕首、長矛、戰士巨劍、暗影之刃 |
| 副手 | `offHand` | 防具/盾 | 木盾 |

### 素質修正機制

裝備物品後，其 `statModifiers` 會修正角色的六大屬性與三大狀態值上限：

```
有效屬性 = 基礎屬性 + Σ(已裝備物品的素質修正)
```

修正值可為正數（增益）或負數（減益）。例如戰士巨劍：力量+4 但敏捷-1。

### 裝備條件檢查邏輯

```swift
func canBeUsedBy(_ character: PlayerCharacter) -> Bool
```

依序檢查：
1. **等階**：`character.circle >= reqCircle`
2. **職業**：`reqGuilds` 為空或包含角色職業
3. **六大屬性**：角色各屬性 >= 對應需求值

## 三、交易數值系統

### 金幣系統

- 角色新建時持有 **100 金幣**（`PlayerCharacter.gold = 100`）
- 金幣為 `Int` 型別欄位，直接儲存在 `PlayerCharacter` 上（SwiftData 持久化）
- 掉落表中的金幣（`01_16_gold_coin`）不進入背包，直接加到 `gold` 欄位

### 交易價格公式（`TradeCalculator`）

#### 購買價（`buyPrice`）

```
購買價 = ceil(物品 value × NPC priceMultiplier)
```

- `value`：物品模板中定義的基礎價值
- `priceMultiplier`：NPC 商品的價格倍率（`NPCShopItem.priceMultiplier`）
- 向上取整（`ceil`）

#### 出售價（`sellPrice`）

```
出售價 = max(floor(物品 value × 0.5 × (1 + 交易技能等級 × 0.01)), 1)
```

- 基礎出售價為物品 value 的 50%
- 交易技能（`SkillType.trading`）每等級增加 1% 售價
- 向下取整（`floor`），最低 1 金幣

### 出售價範例

| 物品 | value | 交易技能 0 級 | 交易技能 5 級 | 交易技能 10 級 |
|------|:-----:|:------------:|:------------:|:-------------:|
| 雞毛 | 1 | 1 | 1 | 1 |
| 兔皮 | 2 | 1 | 1 | 1 |
| 體力草 | 5 | 2 | 2 | 2 |
| 小型治療藥水 | 10 | 5 | 5 | 5 |
| 布靴 | 15 | 7 | 7 | 8 |
| 鐵劍 | 50 | 25 | 26 | 27 |

### NPC 商品定價參考

NPC 商品的 `priceMultiplier` 決定購買價格倍率：

| priceMultiplier | 效果 | 典型用途 |
|:--------------:|------|----------|
| 1.0 | 原價 | 基礎消耗品 |
| 1.2 | 加價 20% | 一般商品 |
| 1.5 | 加價 50% | 較貴商品 |
| 2.0 | 兩倍價 | 稀有商品 |

### 金幣經濟循環

```
金幣來源：
  ├─ 初始金幣：100
  ├─ 哥布林掉落：3~8 金（80% 機率）
  ├─ 沼澤蜥蜴掉落：5~12 金（70% 機率）
  ├─ 山賊掉落：10~25 金（90% 機率）
  ├─ 骷髏戰士掉落：8~18 金（85% 機率）
  ├─ 暗影刺客掉落：15~30 金（90% 機率）
  └─ 石像守衛掉落：20~50 金（95% 機率）

金幣消耗：
  └─ NPC 商店購買物品（村莊商人 + 河谷營地傭兵隊長 + 古代廢墟守望者）
```
