## 三、數值設計分析

- **等級 1 怪物**（兔子、雞）：ATK 1~2 / DEF 0~1 / HP 10~15，適合初入遊戲的角色練習
- **等級 2 怪物**（哥布林）：ATK 5 / DEF 2 / HP 25，人形怪物，主要金幣來源
- **等級 3 怪物**（毒蛙、野豬）：ATK 6~8 / DEF 2~4 / HP 20~40，毒蛙為沼澤入門、野豬為坦克型
- **等級 4 怪物**（礦坑蝙蝠、灰狼）：ATK 7~10 / DEF 2~3 / HP 30~35，蝙蝠為礦坑入門
- **等級 5 怪物**（山賊、沼澤蜥蜴）：ATK 11~12 / DEF 5~6 / HP 50~55，蜥蜴 DEF 高於山賊
- **等級 6 怪物**（沼澤巨蟒）：ATK 14 / DEF 5 / HP 70，沼澤最強怪物
- **等級 7 怪物**（骷髏戰士、暗影刺客）：骷髏為「坦克型」（HP 80/DEF 8），暗影為「刺客型」（ATK 20/HP 60）
- **等級 8 怪物**（石像守衛）：ATK 18 / DEF 12 / HP 120，全遊戲最強怪物，極高防禦與血量

## 四、場景難度分佈

| 場景 | 怪物 | 難度定位 |
|------|------|----------|
| 村莊 | 無 | 安全區（isRestArea） |
| 市集 | 無 | 安全區（isRestArea） |
| 村口 | 無 | 安全區（isRestArea） |
| 後山 | 兔子、雞 | 新手練功區（Lv1） |
| 林間小路 | 兔子、哥布林、野豬 | 過渡區（Lv1~3，金幣來源） |
| 深山 | 野豬、灰狼、山賊 | 進階挑戰區（Lv3~5，高金幣掉落） |
| 河邊渡口 | 無 | 安全區（isRestArea） |
| 河谷營地 | 無 | 安全區（isRestArea，進階商店） |
| 沼澤地帶 | 毒蛙、沼澤蜥蜴、沼澤巨蟒 | 中高階區（Lv3~6，素材掉落） |
| 荒廢礦坑 | 礦坑蝙蝠、骷髏戰士 | 高階區（Lv4~7，金幣+骨片） |
| 古代廢墟 | 暗影刺客、石像守衛 | 最高難度區（Lv7~8，稀有素材+高金幣） |
